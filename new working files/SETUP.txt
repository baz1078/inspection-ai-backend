═══════════════════════════════════════════════════════════════════
  SETUP (5 MINUTES)
═══════════════════════════════════════════════════════════════════

STEP 1: Get API Key
───────────────────
1. Go to: https://console.anthropic.com/
2. Create account (free)
3. Get your API key (it's in the dashboard)

STEP 2: Create .env File
────────────────────────
In inspection-ai-backend directory:

  cp .env.example .env
  
Edit .env and add your API key:

  ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxx
  FLASK_ENV=development

STEP 3: Install Dependencies
─────────────────────────────
  pip install -r requirements.txt --break-system-packages

STEP 4: Run Backend
───────────────────
  python app.py

You should see:
  * Running on http://127.0.0.1:5000
  * Database initialized

  WHAT YOU HAVE
═══════════════════════════════════════════════════════════════════

API ENDPOINTS:

POST /api/upload
  ↳ Upload a PDF inspection report
  ↳ Returns: report_id, share_token, summary

POST /api/ask/<report_id>
  ↳ Ask a question about the report
  ↳ Returns: answer (from report only)

GET /api/conversations/<report_id>
  ↳ Get all Q&A for a report

GET /api/reports
  ↳ List all uploaded reports (admin)

GET /api/shared/<share_token>
  ↳ Get report info via share token (customer portal)

GET /health
  ↳ Verify backend is running

  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

ERROR: "ModuleNotFoundError: No module named 'anthropic'"
FIX: pip install anthropic --break-system-packages

ERROR: "ANTHROPIC_API_KEY not set"
FIX: Check .env file has your key, then restart backend

ERROR: "Cannot connect to backend"
FIX: Make sure "python app.py" is running, check port 5000

ERROR: "PDF extraction fails"
FIX: Try a different PDF, make sure it's readable
